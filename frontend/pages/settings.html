API Example

Create a short link using the API:

POST http://localhost:5000/api/links
Headers:
  Content-Type: application/json
  X-API-Key: your_api_key_here

Body:
{
  "originalUrl": "https://your-long-url.com",
  "customAlias": "mylink",
  "expiresAt": "2025-12-31"
}
